## Project Structure

- `/bin`         : Contient les fichiers exécutables générés lors du build.
- `/build`       : Contient les fichiers temporaires générés pendant le processus de build.
- `/docs`        : Documentation du projet.
- `/include`     : Contient les fichiers d'en-tête (.h).
- `/src`         : Contient les fichiers sources du projet (.cpp).
- `/assets`      : Contient les ressources du projet (textures, modèles, shaders).
    - `config.json` : Fichier de configuration du projet.
- `/tests`       : Contient les tests unitaires.
- `/libs`        : Librairies externes utilisées dans le projet.
- `/third_party` : Librairies tierces utilisées dans le projet.
- `/scripts`     : Scripts d'automatisation pour la configuration ou la construction.
- `CMakeLists.txt` : Fichier de configuration pour le build du projet.
